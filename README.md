# docker-pull-tar-gui

This tool is used for packaging Docker images and can be used out of the box without needing to install any local environment.  
Currently only supports Chinese and English.

此工具用于docker镜像的打包无需本地安装环境开箱即用  
目前仅支持中文和英文
